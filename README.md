# 项目简介

## 核心功能
- 手工制品展示与售卖
- 校园手工活动联动
- 勤工俭学支持
- 安全便捷的交易流程
- 作品评价与改进反馈
- 作品精细化发布
- 制作教程与材料共享
- 闲置物品改造、DIY技能交流、闲置材料互换

## 适用场景
- 校园文化氛围营造
- 学生手工爱好者互动
- 闲置物品再利用
- 学生技能交流平台

## 技术特点
- 采用SpringBoot框架，提供高效且稳定的系统架构
- B/S架构和MVC设计模式，便于系统扩展和维护
- Java、Mysql、SpringBoot、Mybatis、Vue等技术的结合使用，确保系统高性能和易用性
- 可在Windows/Mac操作系统上运行
- 使用IDEA、JDK1.8、Maven、Mysql、node14、navicat等开发环境进行开发
